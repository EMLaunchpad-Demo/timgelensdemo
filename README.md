# Tim Gelens — Kinesitherapie (demo-website)

Een zelfstandige, mobielvriendelijke demo-website voor kinesitherapiepraktijk
Tim Gelens. Geen frameworks, geen externe afhankelijkheden — puur HTML, CSS en
een klein beetje JavaScript.

## Bestanden

| Bestand      | Beschrijving                                  |
|--------------|-----------------------------------------------|
| `index.html` | De volledige pagina (structuur + inhoud)      |
| `style.css`  | Vormgeving en responsive layout               |
| `script.js`  | Kleine interacties (mobiel menu, contactform) |

## Bekijken

Dubbelklik op `index.html` — de site opent in je browser. Werkt ook offline.

## Online zetten met GitHub Pages

1. Ga in deze repo naar **Settings → Pages**.
2. Bij **Source**: kies *Deploy from a branch*.
3. Branch: **`main`**, map: **`/ (root)`** → **Save**.
4. Wacht ~1 minuut. Je website staat dan live op:

   ```
   https://emlaunchpad-demo.github.io/timgelensdemo/
   ```

## Aanpassen

- Teksten, adres en openingsuren staan in `index.html`.
- Kleuren pas je bovenaan `style.css` aan (de `--teal-*` variabelen).
- Vervang telefoonnummer, e-mail en RIZIV-nummer door de echte gegevens.

> Demo — het contactformulier verstuurt nog geen echte e-mail. Koppel het aan
> een formulierdienst (bv. Formspree) om berichten te ontvangen.
